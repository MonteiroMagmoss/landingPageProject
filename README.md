# 🐾 Pequenos Amigos Pet Shop

Landing page moderna desenvolvida para um Pet Shop fictício, com foco em apresentação de serviços, produtos e informações de contato.

## 🚀 Sobre o Projeto

Este projeto foi criado com o objetivo de praticar e demonstrar habilidades em desenvolvimento Front-End, com foco em:

* Estruturação de páginas web
* Estilização com CSS
* Organização de layout
* Interatividade com JavaScript
* Criação de uma experiência visual agradável para o usuário

A landing page apresenta um pet shop completo, incluindo serviços, localização, produtos e contato.
